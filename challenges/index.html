Stego

Steganography is the art of hiding information in different types of medias. It's a very old practice, and invisible ink is an example of how it was used. Nowadays, stego is used to hide message inside digital images or other media files.

Crypto

Cryptography is associated with the process of converting ordinary plain text into unintelligible text and vice-versa. It is a method of storing and transmitting data in a particular form so that only those for whom it is intended can read and process it. It is used in many applications like banking transactions cards, computer passwords, and e- commerce transactions. There are 3 main types of cryptography: Symmetric-key Cryptography, Public-Key Cryptography and Hash Functions.
